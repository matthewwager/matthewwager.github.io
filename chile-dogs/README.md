# Chile Dogs Project Assets

This folder contains the image assets for the Chile Dogs slideshow project.

## Folder Structure

- `images/` - Contains all slide images
  - Background images (SVG)
  - Middleground images (SVG)
  - Foreground images (SVG)

## Naming Convention

Suggested naming pattern:
- `slide-01-background.svg`
- `slide-01-middleground.svg`
- `slide-01-foreground.svg`
- `slide-02-background.svg`
- etc.
